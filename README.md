* [Python 3.5+](https://www.python.org/downloads/)
* [Docker\*](https://www.docker.com/products/docker-desktop)
* [Jupyter\*](https://jupyter.org/install)


\* **Jupyter** and **Docker** are not dependencies/required as of **04/02/20**.
